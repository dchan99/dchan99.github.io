GitHub Jobs API

I want to get data from the GitHub Jobs API and visualize it in some way. Users will be able to type in a keyboard. Then with the data I receieve, I could create some sort of chart, probably a bar graph, that displays the number of jobs by city. I originally planned to create a chart of number of jobs by position of job but each company could rename the job title slightly differently and I don't think it would be possible to combine all relevant job positions.

Example API call: https://jobs.github.com/positions.json?search=python&page=0 Documentation
